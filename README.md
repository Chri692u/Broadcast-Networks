# Modules
### Model
Definitions of process types for broadcast networks:
- Processs types
- Wait-only restriction

### Semantics
Definitions for the semantics of broadcast networks:
- Reliable communication
- Unreliable communication

Semantics consider different topologies:
- Cliques (Reliable and unreliable)
- Arbitrary graphs (Reliable restricted to wait-only and unreliable)
